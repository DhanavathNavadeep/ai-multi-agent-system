# AI Multi-Agent System

## Overview

This project demonstrates a simple Multi-Agent System built using Python. Multiple agents collaborate to perform different tasks in a workflow.

## Agents

### Research Agent

Collects and provides information on a given topic.

### Writer Agent

Uses research data to generate content.

### Reviewer Agent

Reviews the generated content and provides approval.

## Workflow

Research Agent → Writer Agent → Reviewer Agent
